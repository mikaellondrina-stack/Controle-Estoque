# Sistema de Controle de Estoque Porter

Sistema completo de controle de estoque com Firebase em tempo real.

## 🚀 Tecnologias

- Next.js 14 (App Router)
- TypeScript
- TailwindCSS
- Firebase (Auth + Firestore)
- jsPDF para relatórios
- Vercel (Deploy)

## 📋 Pré-requisitos

- Node.js 18+
- Conta no Firebase
- Conta na Vercel (opcional)
